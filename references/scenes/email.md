# 商务邮件场景表达

## 1. 开场与背景铺垫

```
"I'm reaching out because / regarding..."
"I'm writing to follow up on our conversation about..."
"I'd like to update you on the status of..."
"This is a quick note to let you know that..."
```

## 2. 请求类邮件

### 初级 vs 高级

```
❌ "I need you to do something."
✅ "I was hoping you might be able to help with... Would you have bandwidth this week?"

❌ "Please send me the report."
✅ "Would it be possible to get the report by EOD Thursday? I want to make sure I have enough time to review it before the meeting."

❌ "Can you do this ASAP?"
✅ "This is a bit time-sensitive — would it work to get this back by tomorrow afternoon? I can hop on a call if you need to discuss."
```

### 请求模板

```
Subject: Quick favor — [具体请求]

Hi [名字],

Hope your week's going well. I'm reaching out because [背景].

Specifically, I was wondering if you could [具体请求内容]. The reason I'm asking is [原因].

No pressure if you're swamped — just let me know. Happy to jump on a quick call if that's easier. Thanks in advance!
```

## 3. 汇报类邮件

### 项目进展汇报

```
"Here's a quick update on the [项目名] initiative:
- Status: [Green / Yellow / Red]
- Progress: [已完成的内容]
- Next steps: [下一步计划]
- Risks: [如有风险简述]

Happy to discuss in more detail if helpful."
```

### 数据汇报

```
"Sharing the weekly metrics as requested. The highlights:
- DAU is up [X]% WoW, driven by [原因]
- Conversion rate held steady at [X]%
- One thing to flag: [需要关注的事项]

Full data is in the attached dashboard."
```

## 4. 跟进与催单

```
❌ "Please hurry up."（太直接，不够专业）
✅ "I wanted to follow up on this — just checking in to see where things stand. Let me know if the timeline still works or if we need to adjust."

❌ "You didn't reply to my last email."
✅ "I sent a note last [时间] about [主题] — wanted to make sure it reached you. Let me know if you need me to resend or if now's not a good time."

✅ "Just a gentle nudge — no rush, but wanted to keep this on your radar."
```

## 5. 拒绝与委婉推辞

```
"I really appreciate you thinking of me for this, but unfortunately I'm at capacity with [当前项目] right now and wouldn't be able to give it the attention it deserves."

"Thanks for reaching out. My current priorities don't leave much room for [额外任务], so I'll have to pass this time. But keep me in mind for future opportunities!"

"I'd love to help, but this isn't quite in my area of expertise — [推荐更合适的人或方向]. Does that make sense?"
```

## 6. 致谢与结束语

```
"Thanks so much for your help on this — it made a big difference."
"Really appreciate you taking the time to walk me through this."
"Thank you for the update — that's exactly what I needed."
"Thanks for the quick turnaround on this!"
```

### 常用结尾

```
"Let me know if you have any questions."
"Happy to discuss further if helpful."
"Looking forward to hearing from you."
"Let me know what you think — open to feedback."
"Feel free to ping me if anything comes up."
```

## 7. 邮件语气对照表

| 场景 | 不太推荐的表达 | 更地道的表达 |
|------|---------------|--------------|
| 要求对方回复 | "Please reply ASAP" | "Looking forward to your thoughts" |
| 告知坏消息 | "This is bad news..." | "Unfortunately..." / "I have an update..." |
| 说自己很忙 | "I'm very busy" | "I'm at capacity" / "I'm stretched thin" |
| 提醒对方 | "You forgot to..." | "Just a reminder that..." / "Wanted to flag..." |
| 表达同意 | "Yes, that's fine" | "That works for me" / "Sounds good — go ahead" |
| 表示感谢 | "Thank you very much" | "Really appreciate your help on this" |
