# 述职汇报场景表达

## 1. 年度 / 季度述职

### 开场定调

```
"This quarter, I focused on [核心项目], with the goal of [目标]. I'm excited to share what we achieved."
"Over the past [时间段], I've been working primarily on [方向], and I want to walk you through my progress, key learnings, and what's ahead."
"Let me start with a high-level overview of my [时间段]: what I set out to do, what actually happened, and what I've learned."
```

### 成果描述（用数据说话）

```
"I led the [项目名] initiative, which resulted in [具体数据提升], contributing approximately ¥[X]M to [业务指标]."

"One project I'm proud of is [项目名]. The biggest win was [关键结果]. Here's the before and after: [数据对比]."

"We [做了什么事], which led to [结果]. The impact was significant because [业务意义]."
```

### 反思不足

```
"Looking back, one area I'd approach differently is [事情]. If I had to do it over, I would [改进做法], rather than [之前的做法]."

"One thing I'm still working on is [能力/领域]. This quarter showed me I need to [发展方向]."

"Honest assessment: [评估]. My plan to improve is [具体改进计划]."
```

### 未来规划

```
"Next quarter, my focus areas are: first, [方向1]; second, [方向2]; and third, [方向3]."

"For the coming [时间段], I'm prioritizing [项目/能力], with the goal of [目标]."

"My key OKRs for next quarter are: [O1] and [O2], with [具体KR]."
```

## 2. 转正述职

### 新人定位

```
"I'm excited to share my initial thoughts after [X] weeks here. It's been a steep learning curve, and I wanted to reflect on what I've learned and where I'm headed."
```

### 适应与学习

```
"I've spent most of my ramp-up time getting familiar with [产品/系统/流程], and I'm now confident in [具体能力]."

"One thing I've been ramping up on is [知识领域] — I've gone through [资料], talked to [人], and done [实践]."

"I've learned how [具体流程/系统] works here, which is different from my previous experience in [方面]."
```

### 贡献展示

```
"Some of my early contributions include: [贡献1], [贡献2], and [贡献3]. Each of these was a learning experience."

"While still ramping up, I was able to contribute to [项目], which involved [具体做了什么]."

"I took on [任务] early on, and it gave me a chance to [学到什么/产出什么]."
```

### 挑战与应对

```
"The biggest challenge for me was [挑战]. To address it, I [做法]. Looking back, [反思]."

"One area that was harder than expected was [领域]. I've since [解决方案/下一步计划]."
```

### 展望与承诺

```
"For the next [时间段], I'm focused on [方向]. My goal is to [具体目标]."

"I'm excited to take on more ownership in [领域]. Here's my plan: [具体计划]."
```

## 3. 述职常用词汇

| 术语 | 释义 | 用法 |
|------|------|------|
| Drive | 主导、推动 | I drove the redesign of... |
| Deliver | 交付 | We delivered on time and under budget. |
| Scale | 规模化 | We scaled the feature to 10M users. |
| Launch | 上线 | We launched v2.0 in Q3. |
| Onboard | 接手/入职 | I onboarded quickly and started contributing in week 2. |
| Align | 对齐 | We aligned on priorities before kickoff. |
| Ramp up | 适应、上手 | I'm ramping up on the data pipeline. |
| Fall short | 未达标 | We fell short on the retention target. |
| Exceed | 超出 | We exceeded our target by 20%. |
| Prioritize | 排优先级 | We had to prioritize scope under pressure. |

## 4. 述职中常见问题应对

```
Q: "What was your biggest failure this quarter?"
A: "I'd say [事情]. Looking back, [反思]. If I could redo it, I'd [改进做法]. The key lesson I took away was [教训]."

Q: "How do you prioritize when everything is urgent?"
A: "I use a framework: impact vs. urgency. I also check in with my manager to make sure my priorities are aligned with team goals. If push comes to shove, I escalate and ask for help with trade-off decisions."

Q: "What do you need to be more effective?"
A: "Two things: first, more visibility into [信息], so I can make better decisions independently. Second, more mentorship in [领域], where I feel I'm still growing."

## 5. 职场黑话（述职汇报场景）

> 述职时说到"归因"、"定稿"、"上线"等词，老板往往也在用英文——学会这套说法，汇报更地道。

| 中文 | 地道英文 | 释义 | 例句 |
|------|---------|------|------|
| 定稿 | finalized, lock it down | 正式定下来 | The spec needs to be locked down before we start dev. |
| 归因 | attribute to | 归因分析 | We need to attribute the growth to specific campaigns and channels. |
| 打法 | approach, strategy; go-to-market motion | 打法/策略 | What's our go-to-market strategy for next quarter? |
| 抓手 | lever, entry point | 切入点 | The key lever here is improving the checkout flow. |
| 扩规模 | scale up, grow at scale | 扩大规模 | We need to think about how to scale this operation efficiently. |
| 上线 | ship, launch, go live | 发布上线 | We're targeting to ship this by end of month. |
```
