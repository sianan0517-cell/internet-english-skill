# 周末复习 · 本周高频表达回顾

## 本周复习地图

本周我们覆盖了三个核心场景——商务邮件、会议发言、向上沟通。以下是每个场景里最值得记住的表达，按使用频率排列。

---

## 商务邮件（周一）

### 必背 Top 5

**催单跟进**
```
Just wanted to follow up on this — any update on [事项]?
Just checking in. Have you had a chance to look at [事项]?
```

**请求帮助（得体版）**
```
Would you have bandwidth to help with [事项] this week?
I was hoping you might be able to take a look at [事项] when you get a chance.
```

**汇报进度**
```
Quick update: [项目] is at [X]%, on track for [日期].
Here's a status update on [项目]: [摘要]. No blockers right now.
```

**表达感谢**
```
I really appreciate you taking the time to [做某事].
Thanks for your patience on this — I know it took a while.
```

**请求决策**
```
We need a decision on [事项] by [日期] to stay on schedule. Let me know if you need any more info.
```

### 本周新词

- **follow up** — 跟进、追问
- **get a chance** — 有机会（比 "can" 更委婉）
- **on track** — 按计划进行中
- **bandwidth** — 精力/时间余量
- **blocker** — 阻碍因素、卡点

## 会议发言（周二）

### 必背 Top 5

**开场定调**
```
Let's get started. Thanks everyone for joining.
Just to set expectations — I expect we'll need about [X] minutes.
```

**同意观点**
```
I'm on the same page — I think [复述对方观点] makes sense.
That's a great point. I'd add that [补充].
```

**表达保留意见**
```
I see where you're coming from, but I'm not sure [观点] would work because [原因].
I'm a bit concerned about [问题]. Have we considered [替代方案]?
```

**总结讨论**
```
So if I'm hearing this correctly, [总结结论]. Is that right?
To summarize: [结论1] and [结论2]. Shall we move on?
```

**争取发言机会**
```
Can I jump in here? I think [观点] might be relevant.
Can I add something quickly?
```

### 本周新词

- **get started** — 开始
- **set expectations** — 设定预期（说明时间、议程）
- **on the same page** — 看法一致
- **come from** — 来自、引申为"出于...的考虑"
- **jump in** — 插话、加入讨论

---

## 向上沟通（周三）

### 必背 Top 5

**主动预警（风险提前报）**
```
I wanted to flag something before it becomes a bigger issue. [简述问题]. I've already started [应对措施], but wanted to loop you in early.
```

**进度汇报（有数据感）**
```
The [项目] is at [X]% — we're on track to finish by [日期]. No blockers right now.
```

**请求资源（给理由）**
```
Given the scope of this, I'd recommend we allocate [资源]. This would let us hit [目标] without compromising [质量/其他].
```

**表达不确定性（诚实但自信）**
```
I want to flag a potential risk — [问题]. We have a fallback plan, but wanted to make sure you're aware.
```

**提建议（先问再讲）**
```
I have a suggestion that might improve our approach. Would you be open to hearing it, or should I write it up in a doc first?
```

### 本周新词

- **flag** — 标记、提出（警示）
- **loop someone in** — 把某人拉进来知情
- **on track** — 按计划进行
- **fallback plan** — 备用方案、保底方案
- **write it up** — 整理成文档

## 常见错误自查

| 你可能想说 | ❌ 错误版本 | ✅ 正确版本 |
|---|---|---|
| 有时间吗 | Do you have time? | Do you have bandwidth? / when you get a chance |
| 我觉得不行 | I think it can't work | I'm not sure this would work because... |
| 帮我看一下 | Please help me check | Would you be able to take a look at...? |
| 继续 | Continue | Move on / Let's keep going |
| 同意 | I agree | I'm on the same page / That makes sense |
| 跟进 | Follow this | Follow up on this |
