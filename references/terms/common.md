# 通用职场术语（所有岗位适用）

## 一、高频缩写

| 缩写 | 全称 | 释义 | 例句 |
|------|------|------|------|
| ASAP | As Soon As Possible | 尽快 | I need the report ASAP — can you send it over by EOD? |
| EOD | End of Day | 今天结束前 | Please submit your timesheet by EOD today. |
| EOW | End of Week | 本周结束前 | The deliverables are due EOW. |
| ETA | Estimated Time of Arrival | 预计完成时间 | What's your ETA on the report? |
| TBD | To Be Determined | 待定 | The launch date is TBD — we'll announce it soon. |
| TBC | To Be Confirmed | 待确认 | The meeting location is TBC. |
| N/A | Not Applicable | 不适用 | This field is N/A for our use case. |
| FYI | For Your Information | 供你参考 | FYI, the team will be OOO next week. |
| TTYL | Talk To You Later | 稍后聊 | Got a call coming in — TTYL! |
| BRB | Be Right Back | 马上回来 | BRB, grabbing coffee. |
| IMO | In My Opinion | 在我看来 | IMO, we should wait for more data before deciding. |
| IMHO | In My Humble Opinion | 以我愚见 | IMHO, the pricing is too aggressive. |
| ETA | Estimated Time of Arrival | 预估时间 | ETA for the fix is about 2 hours. |
| DM | Direct Message | 私信 | I'll DM you the link. |
| CC | Carbon Copy | 抄送 | CC'ing the legal team on this thread. |
| BCC | Blind Carbon Copy | 密送 | BCC is used when you don't want recipients to see each other's emails. |
| ROI | Return on Investment | 投资回报率 | The ROI for this campaign is 4x. |
| OKR | Objectives and Key Results | 目标与关键成果 | All teams align on OKRs quarterly. |
| KPI | Key Performance Indicator | 关键绩效指标 | Monthly active users is our primary KPI. |
| SLA | Service Level Agreement | 服务水平协议 | We're meeting our SLA of 99.9% uptime. |
| Q&A | Questions and Answers | 问答环节 | We'll have a Q&A at the end of the presentation. |
| WFH | Work From Home | 居家办公 | I'm WFH today — will join the call remotely. |
| OOO | Out of Office | 不在办公室 | OOO next week — ping me on Slack for urgent things. |
| LoY | Length of Tenure | 在职时长 | We look at LoY when considering promotions. |
| QoQ | Quarter over Quarter | 环比 | Revenue grew 8% QoQ. |
| YoY | Year over Year | 同比 | User growth is up 25% YoY. |
| GMV | Gross Merchandise Volume | 成交总额 | GMV for Q3 was ¥10 billion. |
| AOV | Average Order Value | 平均客单价 | AOV increased to ¥150 after the bundle launch. |
| MAU | Monthly Active Users | 月活跃用户 | MAU crossed 50 million this month. |
| DAU | Daily Active Users | 日活跃用户 | DAU is at 5 million — still room to grow. |
| NPS | Net Promoter Score | 净推荐值 | Our NPS improved to 52. |
| CAC | Customer Acquisition Cost | 获客成本 | CAC dropped 15% after optimizing the funnel. |
| LTV | Lifetime Value | 生命周期价值 | LTV is ¥400 — healthy for our business model. |

## 二、会议与协作

| 术语 | 释义 | 例句 |
|------|------|------|
| Sync | 同步会议 | Let's have a quick sync after lunch. |
| Kickoff | 项目启动会 | The kickoff is scheduled for Monday. |
| Standup | 站会 | Standup is at 9:30 every morning. |
| All-hands | 全员大会 | The all-hands is this Friday at 2pm. |
| Town hall | 全体交流会 | We do a town hall every quarter. |
| 1-on-1 | 一对一会议 | I have my 1-on-1 with my manager at 4pm. |
| Retro / Retrospective | 复盘会 | Let's add this to the retro agenda. |
| Wrap-up | 总结收尾 | Let's do a quick wrap-up of what we agreed on. |
| Follow-up | 跟进 | I'll send a follow-up email with the action items. |
| Circulate | 传阅 | Please circulate the deck to all attendees. |
| Loop in | 拉入相关人 | Let me loop in the design team. |
| Loop out | 移出不相关人 | We can loop out the PM from this thread. |
| Escalate | 升级上报 | Should we escalate this issue to management? |
| Unblock | 解除阻碍 | How can I help unblock this? |
| Take offline | 线下讨论 | Let's take this offline — it's getting off-topic. |
| Parking lot | 停车场（话题暂存）| Let's put that in the parking lot and focus on today's agenda. |

## 三、工作状态与流程

| 术语 | 释义 | 例句 |
|------|------|------|
| On track | 按计划进行 | The project is on track — we're hitting milestones. |
| At risk | 有风险 | The launch is at risk due to resource constraints. |
| Off track | 偏离计划 | The project is off track — we need to re-baseline. |
| Blocked | 被阻塞 | We're blocked on the API spec from the backend team. |
| Stalled | 停滞不前 | The initiative has stalled — we need a sponsor. |
| Green / Yellow / Red | 绿/黄/红灯 | Status: Yellow — the data pipeline has delays. |
| Bandwidth | 工作容量 | I don't have bandwidth for this additional task. |
| Capacity | 团队容量 | We're at 90% capacity this quarter. |
| Priority | 优先级 | What's the priority — we can't do everything at once. |
| Scope | 工作范围 | The scope is expanding beyond what was agreed. |
| Scope creep | 范围蔓延 | Scope creep is eating into our timeline. |
| Deadline | 截止日期 | The deadline is next Friday — no flexibility. |
| Turnaround | 响应时间 | What's the turnaround on this report? |
| Lead time | 准备周期 | The lead time for this vendor is 3 weeks. |
| Hand off | 交接 | The design handoff to engineering was smooth. |
| Deliverable | 交付物 | What are the key deliverables for this phase? |
| Ownership | 负责制 | Who owns this initiative? |
| Accountability | 问责制 | We need clear accountability across teams. |
| Buy-in | 认可支持 | We need executive buy-in before moving forward. |
| Go-ahead | 批准推进 | Did we get the go-ahead from leadership? |
| Green light | 绿灯批准 | The project got the green light. |
| Red light | 红灯暂停 | It's a red light until we resolve the budget issue. |

## 四、职场关系

| 术语 | 释义 | 例句 |
|------|------|------|
| Stakeholder | 干系人 | We need to manage stakeholder expectations carefully. |
| Sponsor | 赞助人 | Every initiative needs an executive sponsor. |
| Champion | 倡导者 | We need a champion within each team. |
| Peer | 同级同事 | I talked to my peers about this approach. |
| Direct report | 下属 | She has 8 direct reports. |
| Skip-level | 越级 | Skip-level meetings give leadership visibility into teams. |
| Culture fit | 文化匹配 | Culture fit matters as much as skills. |
| Onboarding | 入职引导 | Good onboarding sets new hires up for success. |
| Ramp-up | 适应期 | New hires reach full ramp-up in 3 months. |
| Performance review | 绩效评估 | The performance review cycle starts next month. |
| Calibration | 绩效校准 | We do calibration to ensure consistent ratings across teams. |
| Promotion | 晋升 | She got a promotion to Senior PM last quarter. |

## 五、职业发展

| 术语 | 释义 | 例句 |
|------|------|------|
| Career path | 职业发展路径 | We offer both IC and management career paths. |
| IC (Individual Contributor) | 个人贡献者 | I'm an IC — I don't manage people. |
| Tenure | 在职年限 | Tenure is one factor in promotion decisions. |
| Leveling | 职级体系 | Our leveling framework goes from L3 to L7. |
| Skill set | 技能组合 | The role requires a different skill set than I currently have. |
| Skill gap | 技能差距 | There's a skill gap we need to address with training. |
| Upskill | 提升技能 | We're investing in programs to upskill the team. |
| Cross-functional | 跨职能 | Cross-functional experience is valuable for leadership roles. |
| Exposure | 曝光度 | This project will give you great exposure to leadership. |
| Mentorship | 导师制 | I'm looking for a mentor who can guide my career growth. |
| Career goals | 职业目标 | What are your career goals for the next 3 years? |
| Lateral move | 平级调动 | A lateral move can broaden your experience. |
| Stretch assignment | 挑战性任务 | The stretch assignment pushed me out of my comfort zone. |
