# 数据分析术语表

## 高频核心词

### 分析方法

| 术语 | 释义 | 例句 |
|------|------|------|
| Cohort Analysis | 队列分析 | Cohort retention looks strong at Day-30. |
| Funnel Analysis | 漏斗分析 | The funnel shows most drop-off at step 3. |
| A/B Test | A/B 测试 | We need more data before calling the A/B. |
| Statistical Significance | 统计显著性 | The lift is 5% but not statistically significant. |
| Correlation | 相关性 | There's a strong correlation between X and Y. |
| Causation | 因果关系 | Correlation doesn't imply causation. |
| Regression | 回归分析 | We ran a regression to predict churn. |
| Distribution | 分布 | The data follows a normal distribution. |

### 指标与数据

| 术语 | 释义 | 例句 |
|------|------|------|
| KPI | 关键绩效指标 | Let's align on the KPIs first. |
| Metric | 指标 | What metric are we optimizing for? |
| Dimension | 维度 | Let's slice by city and device dimension. |
| Granularity | 粒度 | We need higher granularity on this. |
| Variance | 方差 | There's high variance in week-1 data. |
| Outlier | 异常值 | We should exclude outliers from the analysis. |
| Baseline | 基线 | Compare against the baseline. |
| Normalization | 标准化 | Normalize by GMV to remove scale effect. |

### 数据质量

| 术语 | 释义 | 例句 |
|------|------|------|
| Data Quality | 数据质量 | We need to address the data quality issue. |
| Data Lineage | 数据血缘 | What's the data lineage for this metric? |
| Schema | 表结构 | The schema changed last week. |
| ETL | 数据抽取转换加载 | The ETL pipeline runs at 3am. |
| Pipeline | 数据管道 | The pipeline is broken — looking into it. |
| Latency | 延迟 | There's a 24-hour latency in the dataset. |

### 增长分析

| 术语 | 释义 | 例句 |
|------|------|------|
| Retention | 留存 | Day-7 retention is at 35%. |
| Churn | 流失 | Churn spiked after the price change. |
| LTV / CLV | 用户生命周期价值 | LTV is around $150 for this segment. |
| CAC | 客户获取成本 | CAC dropped to ¥30 with the new channel. |
| ARPU | 每用户平均收入 | ARPU is trending up quarter-over-quarter. |
| MRR | 月度经常性收入 | MRR grew 10% MoM. |

## 中级进阶词

| 术语 | 释义 | 例句 |
|------|------|------|
| Dashboard | 数据看板 | Check the dashboard for real-time metrics. |
| Drill-down | 下钻分析 | Let's drill down into the user segment. |
| Attribution | 归因 | What's the attribution model? |
| Trend | 趋势 | There's a clear upward trend. |
| Seasonality | 季节性 | Adjust for seasonality before drawing conclusions. |
| Sample Size | 样本量 | The sample size is too small to be significant. |
| Confidence Interval | 置信区间 | We're 95% confident the true value is within this range. |
| Predictive Model | 预测模型 | We built a predictive model for churn. |
| Feature Engineering | 特征工程 | Feature engineering is the most time-consuming part. |
| Data Governance | 数据治理 | We need better data governance. |
