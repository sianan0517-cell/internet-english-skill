# 设计岗位术语（UI/UX/视觉/交互设计）

## 视觉与界面

| 术语 | 释义 | 用法 |
|------|------|------|
| UI (User Interface) | 用户界面 | The UI needs to be more intuitive for first-time users. |
| Visual design | 视觉设计 | The visual design phase usually comes after wireframing. |
| Typography | 字体排版 | Typography is a key part of brand consistency. |
| Color palette | 色彩方案 | We should stick to the brand's color palette. |
| Iconography | 图标设计 | The iconography needs a refresh — some icons feel dated. |
| Illustration style | 插画风格 | We're going for a more playful illustration style. |
| Layout | 页面布局 | The layout needs to be adjusted for mobile. |
| Grid system | 栅格系统 | We use a 12-column grid system across all pages. |
| Spacing / whitespace | 留白间距 | The whitespace makes the design feel cleaner. |
| Elevation / shadow | 阴影层次 | Elevation helps users understand the hierarchy. |

## UX 与交互

| 术语 | 释义 | 用法 |
|------|------|------|
| UX (User Experience) | 用户体验 | Good UX is about reducing friction at every step. |
| Wireframe | 线框图 | Here's the low-fi wireframe for the new flow. |
| Prototype | 原型 | The prototype is ready for user testing. |
| Mockup | 高保真设计稿 | The mockup looks great, but we need to check edge cases. |
| User flow | 用户流程 | The user flow from landing to checkout is too long. |
| Journey map | 用户旅程图 | The journey map helped us identify pain points. |
| Pain point | 痛点 | One of the main pain points is the login flow. |
| Friction | 摩擦阻力 | We're trying to reduce friction in the onboarding process. |
| Interaction design | 交互设计 | Interaction design focuses on how users engage with the product. |
| Micro-interaction | 微交互 | The micro-interactions on the button make it feel alive. |
| State | 状态（默认/悬停/点击等） | What are all the states we need to design for? |
| Hover state | 悬停状态 | We need a hover state for all interactive elements. |
| Empty state | 空状态 | The empty state should guide users to take action. |
| Error state | 错误状态 | The error state needs to be more user-friendly. |
| Skeleton screen | 骨架屏 | We should use skeleton screens while content loads. |
| A/B test | A/B 测试 | We're A/B testing two versions of the landing page. |

## 设计协作与流程

| 术语 | 释义 | 用法 |
|------|------|------|
| Design system | 设计系统 | We need to document this in the design system. |
| Component | 组件 | This button should be a reusable component. |
| Style guide | 风格指南 | The style guide covers fonts, colors, and usage rules. |
| Brand guideline | 品牌规范 | We need to stay within the brand guideline. |
| Design handoff | 设计交付 | The design handoff to engineering needs to be clearer. |
| Spec / design spec | 设计规格说明 | Can you add specs to the mockup for the dev team? |
| Responsive design | 响应式设计 | Is this design fully responsive across breakpoints? |
| Dark mode | 深色模式 | Dark mode support is a must-have for the next release. |
| Accessibility / a11y | 无障碍设计 | We need to check accessibility compliance. |
| WCAG | 网页内容无障碍指南 | Does this design meet WCAG AA standards? |
| Design review | 设计评审 | Can we schedule a design review for Friday? |
| Iteration | 迭代 | We're on iteration three of the checkout flow. |
| Design debt | 设计债 | We've accumulated some design debt that needs to be addressed. |
| Atomic design | 原子设计 | We're rebuilding the design system using atomic design principles. |
