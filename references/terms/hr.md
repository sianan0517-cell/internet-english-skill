# 人力资源岗位术语

## 招聘与入职

| 术语 | 释义 | 用法 |
|------|------|------|
| JD (Job Description) | 职位描述 | The JD needs to be more specific about the tech stack. |
| Sourcing | 人才寻源 | Our sourcing strategy focuses on two channels. |
| Pipeline | 人才池 | We have a healthy pipeline of candidates for this role. |
| Candidate pool | 候选人库 | We need to expand the candidate pool for diversity. |
| Screening | 简历筛选 | The screening process filters for culture fit. |
| Headcount | 人员编制 | We got approval for two additional headcount. |
| Open req / requisition | 招聘需求 | How many open reqs do we have right now? |
| Offer package | 薪资包 | The candidate accepted our offer package. |
| Onboarding | 入职引导 | Our onboarding program lasts for the first 90 days. |
| Ramp-up | 适应期 | New hires usually reach full ramp-up by month 3. |
| Probation period | 试用期 | The probation period is 6 months. |
| Referral | 内推 | Most of our best hires come through referral. |
| ATS (Applicant Tracking System) | 招聘管理系统 | We manage everything in the ATS. |
| Cold outreach | 主动联系候选人 | The cold outreach response rate was around 15%. |

## 绩效与管理

| 术语 | 释义 | 用法 |
|------|------|------|
| Performance review | 绩效评估 | The performance review cycle is twice a year. |
| Performance rating | 绩效等级 | What rating did you give for this cycle? |
| 360 review | 360度评估 | We're implementing a 360 review process. |
| OKR (Objectives and Key Results) | 目标与关键成果 | The OKR framework keeps teams aligned. |
| KPI (Key Performance Indicator) | 关键绩效指标 | We need to define clear KPIs for each role. |
| Competency | 能力模型 | This role maps to our senior IC competency model. |
| Calibration | 绩效校准 | We need to do a calibration session across teams. |
| Promotion cycle | 晋升周期 | The promotion cycle opens in Q1 and Q3. |
| Leveling | 职级评定 | The leveling framework goes from L3 to L7. |
| Band / salary band | 薪资带宽 | We're at the top of the salary band for this level. |
| Merit increase | 普调 | The merit increase this year is 5%. |

## 组织与员工关系

| 术语 | 释义 | 用法 |
|------|------|------|
| Headcount plan | 人员规划 | The headcount plan for next year is still being finalized. |
| Org design | 组织设计 | Org design decisions need executive approval. |
| Span of control | 管理幅度 | Her span of control is 15 direct reports. |
| Succession planning | 继任计划 | We have a succession plan for all critical roles. |
| Retention | 员工留存 | Retention rate dropped last quarter — we need to investigate. |
| Turnover | 员工流失 | Voluntary turnover is higher than we'd like. |
| Attrition | 自然减员 | The attrition rate for the quarter was 8%. |
| Engagement | 员工敬业度 | We ran an engagement survey last month. |
| L&D (Learning and Development) | 培训发展 | L&D is investing in more leadership programs. |
| PIP (Performance Improvement Plan) | 绩效改进计划 | We put him on a 90-day PIP. |
| Offboarding | 离职交接 | The offboarding process needs to be smoother. |
| Culture fit | 文化匹配度 | Culture fit is just as important as technical skills. |
| DEI (Diversity, Equity, and Inclusion) | 多元公平包容 | DEI initiatives are a company-wide priority. |
