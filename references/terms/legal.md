# 法务合规岗位术语

## 合规与监管

| 术语 | 释义 | 用法 |
|------|------|------|
| Compliance | 合规 | We're ensuring compliance with the new data protection law. |
| Regulatory compliance | 监管合规 | Regulatory compliance is a top priority this year. |
| GDPR (General Data Protection Regulation) | 通用数据保护条例 | GDPR compliance requires explicit user consent. |
| PIPL (Personal Information Protection Law) | 个人信息保护法 | We're reviewing our data practices against PIPL requirements. |
| AML (Anti-Money Laundering) | 反洗钱 | The AML compliance team flagged several transactions. |
| KYC (Know Your Customer) | 客户身份识别 | KYC is mandatory for all new enterprise accounts. |
| Sanctions | 制裁 | We need to check if the vendor is on any sanctions lists. |
| Audit trail | 审计追踪 | Every data access should have a complete audit trail. |
| Risk assessment | 风险评估 | The risk assessment identified three high-priority gaps. |
| Due diligence | 尽职调查 | We're conducting due diligence on the acquisition target. |
| Whistleblower | 举报人 | The whistleblower report is being investigated. |
| Remediation | 整改 | We have 30 days to complete the remediation plan. |

## 数据与隐私

| 术语 | 释义 | 用法 |
|------|------|------|
| Data privacy | 数据隐私 | Data privacy is a growing concern for consumers. |
| Data residency | 数据留存 | Data residency requirements restrict where data can be stored. |
| Data localization | 数据本地化 | Data localization laws require local storage of user data. |
| Consent | 同意 | We need to obtain explicit consent before processing data. |
| Right to erasure | 删除权 | Users have the right to erasure under PIPL. |

| Cross-border data transfer | 跨境数据传输 | Cross-border data transfers require security assessment. |
