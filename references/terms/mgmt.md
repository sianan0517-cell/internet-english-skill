# 管理层 / 通用管理术语

## 战略与决策

| 术语 | 释义 | 用法 |
|------|------|------|
| Vision | 愿景 | Our vision is to be the top platform in the category. |
| Mission | 使命 | The mission statement was refreshed last year. |
| Strategy | 战略 | The strategy for next year focuses on three pillars. |
| Roadmap | 路线图 | Here's the product roadmap for the next 18 months. |
| North Star metric | 核心北极星指标 | Our North Star metric is monthly active users. |
| Playbook | 打法手册 | We need to document the go-to-market playbook. |
| SWOT (Strengths, Weaknesses, Opportunities, Threats) | SWOT 分析 | Let's do a SWOT analysis before the board meeting. |
| Competitive landscape | 竞争格局 | The competitive landscape shifted significantly this year. |
| First-mover advantage | 先发优势 | We want to establish a first-mover advantage in this market. |
| Moat | 护城河 | Our data is our biggest moat against competitors. |
| Pivot | 战略转型 | The company had to pivot after the market shifted. |
| Disrupt | 颠覆 | We're looking for ways to disrupt the market. |

## 组织与执行

| 术语 | 释义 | 用法 |
|------|------|------|
| OKR (Objectives and Key Results) | 目标与关键成果 | All teams are aligned on OKRs this quarter. |
| Alignment | 对齐 | We need to get alignment before moving forward. |
| Buy-in | 认可支持 | We need executive buy-in to proceed. |
| Go-ahead | 批准推进 | Did we get the go-ahead from leadership? |
| Green light | 绿灯/批准 | The project got the green light. |
| Red light | 红灯/暂停 | It's a red light until we resolve the budget issue. |
| Bottleneck | 瓶颈 | Data collection is a bottleneck for the analytics team. |
| Unblock | 解除阻碍 | How can I help unblock this? |
| Milestone | 里程碑 | We're approaching a major milestone. |
| Deliverable | 交付物 | What are the key deliverables for this quarter? |
| Ownership | 负责制 | Who owns this initiative? |
| Accountability | 问责制 | We need clear accountability across teams. |
| RACI (Responsible, Accountable, Consulted, Informed) | 责任矩阵 | Let's map out the RACI for this project. |
| Stakeholder | 干系人 | We need to manage stakeholder expectations. |
| Escalation | 升级上报 | When should we escalate an issue? |
| Governance | 治理 | Data governance is a key concern for the board. |

## 会议与沟通

| 术语 | 释义 | 用法 |
|------|------|------|
| Kickoff | 启动会 | Let's schedule a kickoff for next Monday. |
| All-hands | 全员大会 | The all-hands is this Friday at 3pm. |
| Town hall |  Town hall 会议 | We're doing a town hall to share the strategy update. |
| Sync | 同步会议 | Can we have a quick sync later today? |
| Standup | 站会 | The standup is at 9:30 every morning. |
| Retrospective | 复盘会 | The retrospective highlighted three key improvements. |
| Wrap-up | 总结收尾 | Let's do a quick wrap-up of what we agreed on. |
| Follow-up | 跟进 | I'll send a follow-up email after the meeting. |
| Circulate | 传阅/流转 | Please circulate the deck to all stakeholders. |
| Loop in | 拉入相关人 | Let me loop in the legal team on this. |
| Loop out | 移出无关人 | Let's loop out the PM team from this thread. |
| Cadence | 节奏/规律性 | We have a weekly sync cadence with the executive team. |
