# 产品经理术语表

## 一、需求发现与用户研究

| 术语 | 释义 | 例句 |
|------|------|------|
| User research | 用户研究 | We need to do more user research before finalizing the spec. |
| User interview | 用户访谈 | I conducted 15 user interviews last month. |
| Usability test | 可用性测试 | The usability test revealed three critical pain points. |
| Pain point | 痛点 | We've identified four key pain points in the checkout flow. |
| JTBD (Jobs to Be Done) | 用户的任务需求 | JTBD helps us focus on outcomes, not features. |
| Persona | 用户画像 | We have three primary personas for this product. |
| User story | 用户故事 | As a user, I want to reset my password so that I can regain access. |
| Use case | 用例 | Let's walk through the key use cases together. |
| Gap analysis | 差距分析 | A gap analysis revealed we're missing push notifications. |
| Insight | 洞察 | The key insight from the research: users abandon at step 3. |
| Ethnographic research | 人种志研究 | Ethnographic research helped us understand context of use. |
| Survey | 用户调研 | We ran a survey with 500 respondents. |
| NPS (Net Promoter Score) | 净推荐值 | Our NPS improved from 38 to 52 after the redesign. |
| CSAT (Customer Satisfaction) | 客户满意度 | CSAT is at 4.2 out of 5 — that's solid. |
| VOC (Voice of the Customer) | 客户之声 | We captured a lot of VOC from the beta users. |

| 术语 | 释义 | 例句 |
|------|------|------|

## 四、产品指标

| 术语 | 释义 | 例句 |
|------|------|------|
| DAU (Daily Active Users) | 日活跃用户 | DAU grew 15% this month. |
| MAU (Monthly Active Users) | 月活跃用户 | MAU crossed 20M last quarter. |
| WAU (Weekly Active Users) | 周活跃用户 | WAU gives us a better signal on weekly trends. |
| DAU/MAU ratio | 日活月活比 | DAU/MAU of 0.25 means users are not coming back daily. |
| Stickiness | 用户粘性 | The check-in feature boosted stickiness significantly. |
| Retention | 留存 | Day-7 retention is around 35% — we can do better. |
| Churn rate | 流失率 | Monthly churn rate at 5% — we need to reduce it. |
| GMV (Gross Merchandise Volume) | 成交总额 | GMV grew 30% YoY, reaching ¥5B. |
| Take rate | 平台变现率 | Our take rate is 5%, which is industry standard. |
| ARPU (Average Revenue Per User) | 用户平均收入 | ARPU improved to ¥50 after we introduced premium. |
| ARPPU (Average Revenue Per Paying User) | 付费用户平均收入 | ARPPU is ¥200 — the paying segment is highly valuable. |
| Conversion rate (CVR) | 转化率 | The CVR went from 2% to 3.5% after A/B testing the CTA. |
| Funnel | 转化漏斗 | The funnel shows significant drop-off at the payment step. |
| Cohort analysis | 群组分析 | Cohort analysis shows strong retention for users acquired via referral. |
| LTV (Lifetime Value) | 用户生命周期价值 | LTV is ¥500 — we can afford to spend up to ¥150 on CAC. |
| CAC (Customer Acquisition Cost) | 获客成本 | CAC dropped 20% after we optimized the onboarding flow. |
| LTV/CAC ratio | 生命周期价值/获客成本比 | A healthy LTV/CAC is above 3. |
| Virality coefficient | 病毒传播系数 | The virality coefficient is 1.2 — every user brings in 1.2 new users. |
| Session length | 平均使用时长 | Session length increased from 4 to 7 minutes after the redesign. |
| Time to value (TTV) | 用户价值转化时间 | TTV dropped from 7 days to 3 days after optimizing onboarding. |
